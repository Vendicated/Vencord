## üì• Installation

> ‚ö†Ô∏è **Client mods like EagleCord (Vencord derived) violate Discord‚Äôs Terms of Service.** Use at your own risk ‚Äî most users report no enforcement, but banning is *possible*. 

### Desktop

1. Download the latest EagleCord build from **Releases**.
2. Run the installer or extract and inject EagleCord into Discord.
3. Launch Discord via EagleCord.

### Browser (Userscript/Extension)

1. Install a userscript manager (e.g., Tampermonkey).
2. Add the EagleCord userscript from the Releases page.
3. Reload Discord in your browser.

> *Note:* Browser builds may have limited theme/plugin support compared to the full desktop injector.
