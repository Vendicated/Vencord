# FontStyler for Vencord

This plugin changes how text looks in Discord. If you use it, here is what you can adjust:

- **Custom Fonts** – Paste a Google Fonts URL to replace Discord's default font.
- **Font Size** – Make text larger or smaller using a slider.
- **Font Weight** – Choose how bold or light the text appears.
- **Spacing** – Adjust line height and letter spacing for better readability.
- **Where to Apply** – Toggle whether the new style also affects code blocks and usernames.
- **Reset Button** – One click to restore all settings to Discord's original defaults.

Changes apply instantly as you adjust them. Disable the plugin to return to normal Discord text.
