# ApsenzAppleMusic

Apple Music Rich Presence for Discord on Windows.

This plugin shows the currently playing Apple Music track on Discord with song title, artist, album artwork, and progress bar.

## Why this plugin exists

Vencord already has an Apple Music Rich Presence plugin for macOS. ApsenzAppleMusic focuses on the Windows Apple Music app.

## Features

- Windows Apple Music support
- Song title and artist
- Album artwork when found
- Discord progress bar
- Optional Play on Apple Music button
- Optional clickable song title, artist name, and cover image
- Optional paused display

## Requirements

- Windows
- Apple Music app
- Discord with Vencord

## Recommended settings

- Refresh Interval: 2 or 3 seconds
- Show Artwork: On
- Show Small Icon: Off
- Show Button: Off
- Clickable Song Title: On
- Clickable Artist Name: On
- Clickable Cover Image: On
- Extra Album Line: Off

## Notes

Do not use this plugin together with another Apple Music Rich Presence plugin at the same time.
