# Emote Cloner

![](https://github.com/user-attachments/assets/55e3b011-29bb-4f4d-bf1a-aad94d0f2107)

Allows you to clone emojis and stickers by right-clicking them

You can clone emojis from

-   Onboarding Options
-   Forum Tags
-   User Statuses
-   Emoji Picker
-   In Chat

You can clone stickers from

-   Sticker Picker
-   In Chat
