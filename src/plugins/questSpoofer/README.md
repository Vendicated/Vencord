# Quest Spoofer

A robust automation engine for Discord Quests. It injects a button into the Quests tab that automatically cycles through and completes available quests using the "Stubborn Engine" strategy.

![Visualization](https://i.snipboard.io/0jDT18.jpg)

## Features
*   **Stubborn State:** Maintains persistent presence to guarantee credit accumulation.
*   **Smart Priority:** Automatically prioritizes Videos > Activities > Games > Streams.
*   **Sluggish Cleanup:** Uses a safety buffer after completion to prevent delayed validation errors.
*   **UI Integration:** Adds a native-looking button to the Quests header.
