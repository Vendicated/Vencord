# Channel Name Formatter (Vencord Plugin)

Formats Discord channel names into readable text.

### ✨ What It Does

- Replaces `-` and `_` with spaces  
- Capitalizes each word  

#### Example
```
ingame-chat  →  Ingame Chat
clips_and_media  →  Clips And Media
```
