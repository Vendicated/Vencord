# DarkenStream

A Vencord plugin that allows you to easily lower the brightness of screenshares to combat HDR glare.

## Features
- Adds a "Darken Stream" toggle to the context menu of any screenshare.
- Provides a "Darkness" slider (0-100) to fine-tune the brightness reduction when the feature is enabled.
- Uses CSS filters to dim the stream securely without affecting other Discord elements.
