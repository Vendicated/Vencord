# ChannelTracker

A Vencord plugin that visually tracks your channel interactions in the sidebar.
Originally made to help teams keep better track of Discord ticket systems.

## Features

- **Viewed channels** get an eye icon (👁) next to their name
- **Channels you've messaged in** get a pen icon (✏) next to their name - replaces the eye icon
- Status persists across restarts (stored in IndexedDB via Vencord's DataStore)
