# QuestSpoofer

Advanced quest automation with premium UI and safety features for Vencord.

## Features
- Automated quest enrollment and progress simulation
- Multi-quest queue with concurrency limits
- Notifications and status indicators
- Safety checks and rate-limit handling
- Control panel UI for starting/stopping runs

## Installation
1. Place this folder in `src/userplugins/QuestSpoofer`.
2. Build Vencord or reload user plugins.
3. Enable **QuestSpoofer** in the plugins list.

## Usage
- Open the plugin settings to access the control panel.
- Start/stop spoofing from the UI.
- Optional auto-start can be enabled in settings.
