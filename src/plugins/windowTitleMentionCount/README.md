# Window Title Mention Count

Adds the mention count as a prefix to the window title. Useful for setups where the mention badge isn't visible, such as taskbar setups where the window title is visible but not the badge. Only necessary for Desktop clients, as Vesktop and the website already have a counter appended to the title.

## Example Use Cases

On Windows 10 (and Windows 11 26H2) turn on Small Taskbar Buttons and set Combine taskbar buttons to something other than always hide labels.
Since the taskbar is small, the badge doesn't appear, so we're reliant on the window title.
