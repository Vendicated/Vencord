# mockupEditor

Edit image attachments in Discord before sending.

## Features

- Opens a full editor from the attachment action bar button
- Background modes: transparent, solid, gradient
- Layout controls: padding and scale
- Visual controls: border radius/width/color and shadow
- Annotation tools: text, arrow, rectangle, circle
- Supports multiple image attachments in the same draft

## Usage

1. Attach an image to the message composer.
2. Click the `mockupEditor` paintbrush icon in the attachment action bar.
3. Edit the image and click `Save and Upload`.
4. Send your message.

## Notes

- Transparent background exports to PNG automatically so transparency is preserved.
- Non-editable image formats like GIF/SVG/APNG are ignored.
