# LinkScrub

Automatically removes tracking parameters and cleans links in your messages before sending.

## Features

- **Remove UTM tracking** - Strips cross-site UTM parameters (utm_source, utm_medium, etc.)
- **Remove Social Media trackers** - Strip parameters like `igsh` from URLs which can reveal your account or identity in popups and links.
- **Fix Twitter/X embeds** - Replaces with fxtwitter.com for embedded stats.
- **Fix Instagram embeds** - Instagram notoriously doesn't embed content on Discord so you're forced to login.
- **Shorten links** - Optional shortening for YouTube, Walmart, and Best Buy URLs
- **Smart parameter preservation** - Keeps important params like search queries, video timestamps, and product IDs
- **Unwrap redirects** - Automatically extracts real URLs from Facebook, Google, and other link wrappers
